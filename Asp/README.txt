Dette programmet er kort forklart en interpret, altså et program som leser og analyserer et program i et gitt programmeringsspråk, og som deretter utfører det som dette programmet angir skal gjøres.

Eksempel på kjøring av programmet:
Skriv først "ant" for å kompilere. Deretter:
java -jar asp.jar ukedag.asp

Det er også vedlagt flere andre filer som kan byttes ut med "ukedag" for å teste andre funksjoner i programmet.
